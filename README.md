# Student Grade Tracker (Java)

This project is a console-based Java application developed as part of the CodeAlpha internship task.

## Features
- Takes student marks as input
- Stores marks using ArrayList
- Calculates average score
- Finds highest and lowest marks
- Displays a summary report

## Technologies Used
- Java
- ArrayList
- Scanner

## How to Run
1. Compile the program using:
   javac StudentGradeTracker.java
2. Run the program using:
   java StudentGradeTracker

## Outcome
This project helped me understand Java basics such as loops, conditional statements, and data handling using collections.
